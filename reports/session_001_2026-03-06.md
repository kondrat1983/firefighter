# Session 001 — 2026-03-06

## Goal
Build initial Firefighter QA dashboard — live game health monitoring tool that reads real Steam reviews and surfaces active issues.

## What Was Done

### Backend (`demo_backend.py`)
- Built Flask-based demo backend on port `8001`
- Integrated real **Steam Reviews API** (`store.steampowered.com/appreviews/{appid}`) — no key needed
- 10 games monitored: DDV, Fortnite, Among Us, OW2, Cyberpunk, Elden Ring, GTA V, Helldivers 2, Palworld, Apex
- Keyword-based issue classification into 7 types: `crash`, `connectivity`, `performance`, `exploit`, `progression`, `sentiment`, `bug`
- `build_real_alerts()` — groups Steam reviews by `(game, issue_type)`, requires ≥2 reviews per group to create an alert
- Confidence formula: `min(0.97, 0.50 + len(reviews)*0.04 + total_relevance*0.008)`
- Alert detail endpoint (`/api/alerts/{id}`) returns full evidence, timeline, AI summary, suggested investigation steps
- Game detail endpoint (`/api/games/{id}`) returns game info, filtered alerts, steam_summary, issue_breakdown

### Frontend
- Dashboard page (`/`) with Threat Assessment Grid, Active Alerts feed, Live Stream panel
- Game detail page (`/games/[id]`) — health ring SVG, stat cards, alert list, issue breakdown bars, Steam Stats
- Alert detail page (`/alerts/[id]`) — AI Analysis, Evidence (real Steam reviews), Investigation Steps, Timeline, QA Actions

### Key Fixes
- Hydration errors from date formatting → `suppressHydrationWarning` + `unix * 1000`
- Alert timestamp: `time.time()` returns seconds, JS `Date()` needs ms → `* 1000`
- "Investigate →" overflow → `min-w-0` left, `flex-shrink-0 whitespace-nowrap` right
- YouTube iframe showing ads → replaced with thumbnail CDN + play button overlay
- Stream selector button overflow → `flex-shrink-0 w-20` + `overflow-x-auto`
- Removed bottom panels: Game Health Status, Patch Monitoring, System Metrics

## State at End of Session
- 15 real alerts from real Steam reviews across 10 games
- Bug found (carried to next session): evidence card external link opens store page instead of reviews section
